# Al Hama — Luxury Automotive Brand Website

A professional website built with **Next.js** (frontend) and **Strapi** (CMS).

## 🎯 Project Structure

```
al-hama-website/
├── frontend/          # Next.js application
├── cms/               # Strapi CMS backend
└── docs/              # Documentation
```

## 🚀 Quick Start

### Frontend
```bash
cd frontend
npm install
npm run dev
# Open http://localhost:3000
```

### CMS Backend
```bash
cd cms
npm install
npm run develop
# Open http://localhost:1337/admin
```
